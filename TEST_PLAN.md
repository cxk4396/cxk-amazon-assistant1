## 测试用例清单

### TC01: 应用启动测试
**优先级**: P0 | **类型**: 功能测试

#### 测试步骤
1. 运行 `npm run tauri-dev`
2. 观察应用启动过程
3. 检查主窗口显示

#### 预期结果
- 应用窗口正常打开
- 标题栏显示 "CXK运营决策助手"
- 左侧导航菜单完整显示6个菜单项
- 默认显示首页内容

---

### TC02: 系统设置测试
**优先级**: P0 | **类型**: 功能测试

#### 测试步骤
1. 点击左侧"系统设置"
2. 配置AI API:
   - Base URL: `https://api.timebackward.com/v1`
   - API Key: [测试密钥]
   - 模型: Claude Sonnet 4.6
3. 点击"测试连接"
4. 修改分析参数
5. 点击"保存设置"

#### 预期结果
- 设置页面正常显示
- API配置表单可正常填写
- 测试连接返回成功提示
- 设置保存成功，刷新后配置保留

---

### TC03: 文件上传测试 - AM67销售数据
**优先级**: P0 | **类型**: 功能测试

#### 测试步骤
1. 进入"数据上传"页面
2. 点击AM67卡片的"选择文件"
3. 选择 test_am67.xlsx
4. 等待解析完成
5. 点击"选择列"
6. 在列选择器中选择部分列
7. 点击"确认选择"

#### 预期结果
- 文件选择对话框正常打开
- 文件解析成功，显示"已上传 3 条"
- 列选择器弹出，显示所有列名
- 列选择后成功保存
